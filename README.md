# Wishy-Wash

Wishy-Wash is a simple app concept I designed using **Figma** as part of an exercise in **Codedex**.  
The project was later reposted on **LinkedIn :)**, which made me really proud to see my design shared!

## ✨ Features
- Clean and minimal interface
- Designed entirely in Figma
- Focused on practicing UI/UX fundamentals

## 🎯 Purpose
This project was created as a learning exercise to:
- Explore design thinking
- Practice prototyping in Figma
- Share creative work with the Codedex community

## 📌 Notes
This is not a fully functional app — it’s a design prototype.  
The goal was to experiment, learn, and have fun with the process.
